# jjnicolas.github.io

Landing page hub for my macOS apps and developer tools, published with GitHub Pages
at **https://apps.tnfnet.org**.

It's a static site — plain `index.html` + `style.css`, no build step. To add a new
app, copy an existing `<article class="card">` block in `index.html` and update the
icon, name, description, tags, and links.
